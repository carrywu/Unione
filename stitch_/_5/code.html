If f(x) = x² - 4x + 3, what is the minimum value of the function?
A -1 check_circle
B 0 cancel
C 1
D 3

lightbulb Concept Refresher

The minimum value of a downward-opening parabola y = ax² + bx + c occurs at x = -b/(2a). Here, x = 4/2 = 2. Substituting x=2 back into f(x) gives (2)² - 4(2) + 3 = 4 - 8 + 3 = -1.

In Shakespeare's "Macbeth", what is the primary thematic significance of the repeated motif of sleep?
A It symbolizes physical exhaustion from warfare.
B It represents innocence, peace of mind, and clear conscience. check_circle
C It acts as a gateway to supernatural prophecies. cancel
D It signifies the vulnerability of kings to assassination.

lightbulb Concept Refresher

Macbeth claims he has "murdered sleep," highlighting how his guilt deprives him of the natural rest granted to the innocent. While prophecies occur, sleep itself is fundamentally tied to moral peace.
