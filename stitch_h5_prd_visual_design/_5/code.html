本题考查的是基期比重的计算。根据材料可知，2022年全国规模以上工业企业实现利润总额84038.5亿元，比上年下降4.0%。其中，制造业实现利润总额64150.2亿元，下降13.4%。基期比重 = (部分量 / 整体量) * [ (1 + 整体增长率) / (1 + 部分增长率) ]。代入数据计算可得结果约为77.4%。

Q2. cancel 错误

你的答案 C
正确答案 A

lightbulb 解析

本题属于选词填空。空缺处所在句子意为“面对复杂多变的国际形势，我们必须保持（ ）的头脑”。“清醒”指头脑清楚，符合语境；“理智”侧重于不冲动；“冷静”侧重于不慌乱。结合上下文，强调的是在复杂局势中看清方向，故选A更为准确。错选C可能是因为忽略了“复杂多变”对认知层面的要求。
